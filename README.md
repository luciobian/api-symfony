## API Symfony

Curso de udemy: **PHP Symfony 4 API Platform + React.js Full Stack Masterclass**

## Instalación

```
git clone https://github.com/luciobian/api-symfony.git
```

### Dependencias

```
cd <project-folder>
composer install
```

### Migración 

```
php bin/console doctrine:migrations:migrate
```
